## Add a CA Certificate
{% entity_examples %}
entities:
  ca_certificates:
    - cert: |
        -----BEGIN CERTIFICATE-----
        CERTIFICATE_CONTENT
        -----END CERTIFICATE-----

{% endentity_examples %}

## Enable the mTLS plugin

{% entity_examples %}
entities:
  plugins:
  - name: mtls-auth
    service: example-service
    config:
      send_ca_dn: true
      ca_certificates:
      - ${cert}
variables:
  cert:
    value: $CA_CERTIFICATE_ID
{% endentity_examples %}

## Create a Consumer

[Consumers](/gateway/entities/consumer/) let you identify the client that's interacting with {{site.base_gateway}}. For mutual TLS authentication, each Consumer needs a UUID.

{% entity_examples %}
entities:
  consumers:
    - username: alex
      mtls_auth_credentials:
        - id:
          subject_name:
{% endentity_examples %}
