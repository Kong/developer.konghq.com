## Create a `KongKey`

It will make it part of the Gateway Control Planeâ€™s configuration.

spec:

## Create a `KongKeySet`

Provision a Key-set to logically group related keys.

<!-- vale off -->
{% konnect_crd %}
kind: KongKeySet
apiVersion: configuration.konghq.com/v1alpha1
metadata:
  name: key-set
spec:
  controlPlaneRef:
    type: konnectNamespacedRef
    konnectNamespacedRef:
      name: gateway-control-plane
  name: key-set
{% endkonnect_crd %}
<!-- vale on -->
