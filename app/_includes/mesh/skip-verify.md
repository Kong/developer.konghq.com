{:.warning}
> The `--skip-verify` flag turns off TLS verification of the control plane certificate and is only appropriate for testing purposes, where the control plane uses a self-signed certificate. `kuma-dp` verifies the control plane certificate by default. In production, pass the control plane CA with `--ca-cert-file=/path/to/ca.pem` or set the `KUMA_CONTROL_PLANE_CA_CERT_FILE` environment variable instead of skipping verification.
